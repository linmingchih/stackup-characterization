# Stackup Table Viewer

`stackup_viewer.html` 檔案是一個用戶端網頁應用程式，旨在視覺化和操作來自 XML 檔案的疊層資料。

以下是其功能的詳細說明：

*   **檔案輸入：** 允許使用者上傳包含疊層定義的 XML 檔案。
*   **XML 解析：** 解析上傳的 XML 以提取材料屬性（Dk、Df）和層特定屬性（名稱、類型、厚度、蝕刻因子、表面粗糙度參數）。它識別訊號層並確定其參考層（上方和下方）。
*   **互動式表格顯示：** 將提取的疊層資料呈現在一個可編輯的 HTML 表格中。使用者可以直接修改表格儲存格中的值（層名稱除外）。
*   **可配置參數：** 包含用於全域「頻率」設定的輸入欄位，以及用於特定欄位標頭（例如，蝕刻因子、厚度、Dk、Df、目標阻抗、目標損耗）的「變化」或「容差」百分比。這些允許使用者定義疊層的敏感度參數。
*   **JSON 匯出：** 允許使用者將表格資料的當前狀態，包括頻率和所有配置的變化/容差設定，下載為 JSON 檔案。
*   **樣式：** 提供基本的 CSS，用於清晰、可讀的表格佈局，並為變化和容差欄位提供獨特的樣式。
*   **錯誤處理：** 顯示友善的使用者訊息，用於無效的 XML 檔案或檔案讀取錯誤。

總之，這個 HTML 頁面作為一個互動式工具，供工程師快速檢查、調整和匯出疊層配置，可能用於進一步的分析或模擬。
